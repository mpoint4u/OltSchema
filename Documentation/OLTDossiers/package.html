Name: Package
Containing Schema: DossierItem.xsd
MinOccurs 0
MaxOccurs (1)
Abstract False
Documentation: 
A Package DossierItem is used to group items. Only one level of recursion is allowed. An example of a package is a PNR, or a TOSV, or a CMIR. The Package generally contains the price information of all elements of the package.
